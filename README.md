## Työn aihe

Työn aiheena on aihevalintalistaus opintotöille. Käyttäjälle näytetään lista aiheista
joista hän voi valita yhden. Aiheiden yhteydessä pidetään kirjaa siitä kuinka paljon
kutakin aihetta on valittu.
